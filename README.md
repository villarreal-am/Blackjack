# Blackjack

## Overview📖
A game of casino Blackjack playable inside a terminal window. You and the dealer are each dealt cards, and whoever gets the closest score to 21 wins the game. Scores are printed to the terminal after each new card is dealt (aside from the inital card dealings) for conoenience. To play, run the file through a terminal window. You should be prompted to type "START" to play. Good luck.
